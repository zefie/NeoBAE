Usage

Include the required scripts in your HTML:

<script src="./player-class.js"></script>
<script src="./engine.js"></script>
<script src="./miniplayer.js"></script>

Then simply call:

playSong('path/to/song.mid', 'path/to/soundbank.hsb');
Note: The player is self-contained and handles all UI, controls, and playback. No additional HTML or CSS needed!

API Reference

playSong(songSource, bankSource, options)

Parameters:

  • songSource - URL or ArrayBuffer of MIDI/RMF file
  • bankSource - URL or ArrayBuffer of soundbank (.hsb, .sf2, etc.)
  • options (optional) - Configuration object:
    • title - Custom title for player (default: filename)
    • autoplay - Auto-play on load (default: true)

Example:

playSong('song.mid', 'bank.hsb', {
    title: 'My Amazing Song',
    autoplay: false
});

closeMiniPlayer()

Programmatically close the player:

closeMiniPlayer();
